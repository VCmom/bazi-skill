## 核心算法

### Step 1：判断日主强弱

统计四柱（年月日时）天干地支藏干的五行力量：

| 柱 | 权重 |
|----|------|
| 年柱 | 0.8 |
| 月柱 | **1.5**（月令权重最大）|
| 日柱 | 1.0 |
| 时柱 | 0.8 |

- 天干计 1.0 权重
- 地支藏干：本气 0.6 / 中气 0.3 / 余气 0.1

**失令**（月支五行 ≠ 日主五行）→ 总分 × 0.8 折扣

**判定**：
- 总分 ≥ 4.0 → 旺
- 总分 ≥ 2.5 → 中和
- 否则 → 弱

### Step 2：判定喜忌神

| 日主 | 喜神 | 忌神 |
|------|------|------|
| **身旺** | 食伤（我生）+ 财（我克）+ 官杀（克我）→ 泄耗克 | 印（生我）+ 比劫（同我）→ 生扶 |
| **身弱** | 印（生我）+ 比劫（同我）→ 生扶 | 食伤 + 财 + 官杀 → 泄耗克 |
| **中和** | 视具体组合而定 | — |

**五行映射**：
- 食伤 = 我生 = 木→火 / 火→土 / 土→金 / 金→水 / 水→木
- 财 = 我克 = 木→土 / 火→金 / 土→水 / 金→木 / 水→火
- 官杀 = 克我 = 反向
- 印 = 生我 = 反向
- 比劫 = 同我

### Step 3：打分公式

对每个时间干支 (天干, 地支) 打分：

| 因素 | 权重 | 说明 |
|------|------|------|
| 天干 vs 日主喜忌 | 1.0 | 喜 +1，忌 -1 |
| 地支本气 vs 喜忌 | 1.0 | 喜 +1，忌 -1 |
| 地支中气 vs 喜忌 | 0.3 | 喜 +0.3，忌 -0.3 |
| 地支余气 vs 喜忌 | 0.1 | 喜 +0.1，忌 -0.1 |
| 与日支六合 | +0.5 | 子丑、寅亥、卯戌、辰酉、巳申、午未 |
| 与日支六冲 | -1.0 | 子午、丑未、寅申、卯酉、辰戌、巳亥 |

夹紧到 [-3, +3] 整数。

### Step 4：分→星 映射

```
stars = 3 + score × 2 / 3    # 线性映射
```

| 分数 | 星数 | 含义 |
|------|------|------|
| +3 | 5 星 | 大吉 |
| +1.5 ~ +2.5 | 4-4.5 星 | 吉利 |
| -0.5 ~ +0.5 | 2.5-3.5 星 | 平稳 |
| -1.5 ~ -2.5 | 1.5-2 星 | 不利 |
| -3 | 1 星 | 大凶 |

## 运势 4 维度

### 1. 今日运势

**输入**：今日日柱（公历 → sxtwl.getDayGZ）

**计算**：
- 基础分 = score_gz_to_chart(今日干支, 八字)
- 整体 = score_to_stars(基础分)
- 事业/财运/感情/人际 = 基础分 + 十神 bonus ± 0.5

**十神 Bonus 表**：

| 十神 | 事业 | 财运 | 感情 | 人际 |
|------|------|------|------|------|
| 正官 | +0.5 | | | |
| 七杀 | +0.5 | | | -0.5 |
| 食神 | | +0.5 | | +0.5 |
| 伤官 | +0.5 | | | -0.5 |
| 正财 | | +0.5 | +0.5 | |
| 偏财 | | +0.5 | | |
| 比肩 | | | | +0.5 |
| 劫财 | | -0.5 | | +0.5 |
| 正印 | +0.5 | | | |
| 偏印 | +0.5 | | | |

### 2. 本周运势

**输入**：本周一到周日的 7 个日柱

**计算**：
- 平均分 = 7 天 score 之和 ÷ 7
- 星数 = score_to_stars(平均分)
- 最吉日 = 7 天中分最高
- 最凶日 = 7 天中分最低

### 3. 本月运势

**输入**：本月月柱 + 流年

**计算**：
- 月柱分 = score_gz_to_chart(月柱, 八字)
- 流年分 = score_gz_to_chart(流年, 八字)
- 综合分 = 月柱分 × 0.6 + 流年分 × 0.4
- 星数 = score_to_stars(综合分)

### 4. 本年运势

**输入**：本年流年（暂未叠加当前大运）

**计算**：
- 分数 = score_gz_to_chart(流年, 八字)
- 星数 = score_to_stars(分数)

## 健康 4 维度

### 五行 → 脏腑 映射

| 五行 | 脏 | 腑 | 部位 |
|------|-----|-----|------|
| 木 | 肝 | 胆 | 目/筋/爪甲 |
| 火 | 心 | 小肠 | 舌/血脉/面部 |
| 土 | 脾 | 胃 | 口/唇/肌肉 |
| 金 | 肺 | 大肠 | 鼻/皮肤/毛发 |
| 水 | 肾 | 膀胱 | 耳/骨骼/头发 |

### 状态判定

| 五行 vs 喜忌 | 状态 | 含义 |
|-------------|------|------|
| 喜 | 得助 | 该脏腑气充足 |
| 忌 | 受制 | 该脏腑气偏亢，需减压 |
| 其他 | 平和 | 顺其自然 |

### 1. 今日健康

- 今日日支 → 五行 → 脏腑
- 状态：得助 / 受制 / 平和

### 2. 节气健康

- 当前 24 节气（按月日近似判断）
- 节气五行 → 脏腑
- 状态：得助 / 受制 / 平和

### 3. 季节健康

- 春（3-5月）→ 木 → 肝/胆
- 夏（6-8月）→ 火 → 心/小肠
- 秋（9-11月）→ 金 → 肺/大肠
- 冬（12-2月）→ 水 → 肾/膀胱

### 4. 年度健康

- 流年天干 → 五行 → 脏腑
- 状态：得助 / 受制 / 平和

## 措辞规范

### ❌ 禁止
- "受克" / "受冲" → **恐吓性**
- "心气受冲" → 听起来像心脏有问题
- "血光之灾" / "大凶" → 极端断语
- "必" / "一定" → 绝对化

### ✅ 推荐
- "受制" / "偏亢" → 中性
- "得助" / "充足" → 积极
- "建议" / "宜" → 引导性
- "可能" / "倾向于" → 模糊化
- "以医学诊断为准" → 健康警示

### 健康建议模板

| 状态 | 模板 |
|------|------|
| 得助 | `{脏}气充足，{部位}状态良好，适度活动即可，避免过补` |
| 受制 | `{脏}气偏亢，{部位}需减压，建议清淡饮食、规律作息` |
| 平和 | `{脏}气平稳，{部位}无需特别调理，顺其自然` |

### 运势建议模板

| 维度 | 模板 |
|------|------|
| 5 星 | `大吉日/周/月/年，{动作}，{效果}` |
| 4 星 | `吉日/周/月/年，{动作}可成` |
| 3 星 | `平稳，{动作}，{风险提示}` |
| 2 星 | `不利，{建议规避}` |
| 1 星 | `大凶，{建议休息/保守}` |

## 算法局限性

1. **节气日期精度**：当前实现按月日阈值近似，±1-2 天偏差
2. **大运交互**：本年运势暂未叠加当前大运
3. **空亡 / 神煞**：未实现（暂不纳入）
4. **流月细粒度**：本月运势以月柱为单位，未细化到每旬
5. **起运年龄**：本算法只算当前大运，暂不计算具体流年交接

如需更高精度，建议参考 `references/classical-texts.md` 中 9 本典籍的进阶规则。
